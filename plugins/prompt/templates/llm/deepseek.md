# DeepSeek-R1 / DeepSeek Reasoning Models

## Critical Rule

Reasoning-native like o1. **Do NOT add CoT instructions.**

## Best Practices

- Short clean instructions only
- State goal and desired output format
- Strong at math, code, and logical reasoning

## Output Behavior

Outputs reasoning in `<think>` tags by default. If you only want the answer:
> "Output only the final answer, no reasoning."

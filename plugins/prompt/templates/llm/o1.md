# o1 / o3 / OpenAI Reasoning Models

## Critical Rule

**NEVER add Chain of Thought instructions.** These models reason internally across thousands of tokens. Adding "think step by step" actively degrades output.

## Best Practices

- SHORT clean instructions ONLY
- State what you want, not how to think about it
- Keep system prompt under 200 words
- Define goal and success criteria only
- Trust the model to reason

## Avoid

- CoT scaffolding ("think through this...")
- Heavy XML structure
- Multi-level formatting
- Long system prompts

## Format

```
[Direct goal statement]
[Success criteria]
[Output format]
```

That's it. Nothing else.

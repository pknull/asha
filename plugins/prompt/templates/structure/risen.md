# RISEN Template

Role, Instructions, Steps, End Goal, Narrowing. Use for complex projects, multi-step tasks, clear sequences.

## Structure

```
Role: [Expert identity]
Instructions: [Overall task in plain terms]
Steps:
  1. [First action]
  2. [Second action]
  3. [Continue as needed]
End Goal: [What final output must achieve]
Narrowing: [Constraints, scope limits, exclusions]
```

## Example

```
Role: Product manager with 10 years mobile app experience.
Instructions: Write product requirements document for habit tracking feature.
Steps:
  1. Define problem statement in one paragraph
  2. List user stories: "As a [user], I want [goal] so that [reason]"
  3. Define acceptance criteria for each story
  4. List out-of-scope items explicitly
End Goal: PRD that engineering team can begin sprint planning from immediately.
Narrowing: No technical implementation details. No wireframes. Under 600 words.
```

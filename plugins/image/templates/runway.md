# Runway Gen-3 Video Prompt Template

## Format Rules

- Responds to cinematic language
- Reference film styles or directors for consistent aesthetic
- Camera movement + subject + action + environment

## Structure

```
[Camera style]: [Subject] [action] in [environment]. [Mood/atmosphere]. [Lighting style]. [Film reference if applicable].
```

## Example

```
Tracking shot, low angle: Massive troll warrior rides a dire wolf through a fog-laden swamp. Dead trees pass in foreground. Dark fantasy atmosphere, desaturated color palette with occasional bioluminescent accents. Diffused overcast lighting. Reminiscent of del Toro creature films.
```

## Film Reference Examples

- "Reminiscent of Blade Runner 2049" — orange/teal, hazy, slow
- "In the style of Wes Anderson" — symmetrical, pastel, whimsical
- "del Toro creature aesthetic" — dark, organic, detailed monsters
- "Fincher-esque" — dark, desaturated, precise

## Best Practices

- Cinematic language over technical specs
- Describe movement and action
- Use film/director references for style lock
- Include atmosphere and mood

# DALL-E 3 Prompt Template

## Format Rules

- Prose description works well — natural language understood
- Add "do not include text in the image" unless text needed
- For complex compositions: describe foreground, midground, background separately

## Structure

```
[Subject description in natural prose]. [Setting and environment]. [Style and mood]. [Lighting and color]. [Composition notes]. Do not include any text in the image unless explicitly specified.
```

## Example

```
A weathered troll warrior with mottled green skin and bone armor rides a massive dire wolf through a murky swamp. Twisted dead trees emerge from black water, with fog rolling across the surface. Dark fantasy art style with muted earth tones and hints of bioluminescence. Dramatic low-angle composition with cinematic lighting from above. Do not include any text in the image.
```

## Best Practices

- Be specific about subject details
- Describe spatial relationships clearly
- Specify art style explicitly
- Include lighting direction and quality
- Always add the "no text" clause

## Common Mistakes

- Assuming it understands vague references
- Not specifying composition
- Forgetting to exclude text

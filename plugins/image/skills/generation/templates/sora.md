# Sora Video Prompt Template

## Format Rules

- Describe scene as if directing a film shot
- Camera movement is critical — static vs dolly vs crane changes output dramatically
- Include duration, mood, cut style
- Specify subject continuity across frames

## Structure

```
[Camera movement]: [Subject description] [action]. [Setting]. [Duration] seconds. [Mood/atmosphere]. [Lighting]. [Style reference if applicable].
```

## Camera Movements

| Movement | Effect |
|----------|--------|
| Static shot | Fixed camera, subject moves |
| Dolly in/out | Camera moves toward/away |
| Tracking shot | Camera follows subject |
| Crane up/down | Vertical camera movement |
| Pan left/right | Camera rotates horizontally |
| Handheld | Organic, slight shake |

## Example

```
Slow dolly forward: A troll warrior on a dire wolf emerges from fog in a murky swamp. Camera at low angle, moving through twisted dead trees toward the subject. 8 seconds. Dark fantasy atmosphere, muted greens and browns with bioluminescent accents. Overcast lighting with volumetric fog. Cinematic film grain.
```

## Best Practices

- Lead with camera movement
- Specify duration in seconds
- Describe motion/action, not static scene
- Include atmosphere and lighting
- Reference film styles for consistent aesthetic
